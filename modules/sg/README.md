# Security Group Module

This module provides a reusable and configurable way to create and manage security groups in your cloud infrastructure. It allows you to define ingress and egress rules, associate security groups with resources, and customize various settings to meet your security requirements.

## Features

- Create security groups with customizable names and descriptions.
- Define ingress and egress rules with specific protocols, ports, and CIDR blocks.
- Associate security groups with instances, load balancers, and other resources.
- Support for tagging and metadata.
- Integration with other infrastructure modules.
